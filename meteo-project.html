Contexte du projet

Pourquoi ce TP ?

🎓

Objectif pédagogique

TP BTS SIO SLAM visant à concevoir un agent conversationnel capable d'interpréter des questions en langage naturel et de retourner des données réelles via des API externes.

🧠

IA intégrée

Mistral AI (modèle français 🇫🇷) analyse la question de l'utilisateur, extrait le nom de la ville et formule une réponse naturelle autour des données météo récupérées.

🌍

API Temps réel

OpenWeatherMap fournit la température, le ressenti, l'humidité et la vitesse du vent pour n'importe quelle ville dans le monde, en temps réel.

🔒

Conformité CNIL

L'application affiche une bannière de transparence, ne stocke aucune conversation et minimise les données traitées au strict nécessaire (nom de ville uniquement).

Fonctionnalités

Ce que fait l'application

01

Langage naturel

L'utilisateur écrit librement : "Quel temps fait-il à Lyon ?" ou "Météo à Marseille". L'IA comprend et extrait la ville automatiquement.

02

Carte météo détaillée

Température, ressenti, humidité et vitesse du vent affichés dans une carte visuelle intégrée dans la bulle de réponse du chatbot.

03

Indicateur de frappe

Animation "L'agent réfléchit…" affichée pendant l'appel API pour informer l'utilisateur que la requête est en cours de traitement.

04

Envoi par touche Entrée

Support du raccourci clavier Enter pour envoyer un message, en plus du bouton, pour une expérience fluide et naturelle.

05

Gestion des erreurs

Villes inconnues ou problèmes de connexion gérés proprement avec un message d'erreur explicite dans le chat.

06

Bannière CNIL

Information obligatoire affichée en haut de page indiquant que l'utilisateur interagit avec un agent automatique.

Fonctionnement

De la question à la réponse

1
Question

L'utilisateur saisit une question en langage naturel

2
Mistral AI

L'IA extrait le nom de la ville depuis la question

3
API Météo

Flask interroge OpenWeatherMap avec la ville extraite

4
Réponse IA

Mistral formule une réponse naturelle avec les données

5
Affichage

Carte météo + texte affiché dans le chat

Conformité

🔒 Respect des recommandations CNIL

✓
Transparence

L'utilisateur est informé qu'il interagit avec un robot, pas un humain.

✓
Minimisation

Seul le nom de la ville est traité. Aucune donnée personnelle collectée.

✓
Pas de stockage

Aucune conversation n'est enregistrée. Traitement en temps réel uniquement.

✓
Sécurité

Les clés API sont stockées côté serveur et jamais exposées au client.
